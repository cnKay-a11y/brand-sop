01视觉策略推导

翻译官 · 将文字策略转化为视觉关键词
🎨

1.1 情绪板制作

  • 收集图片 / 色彩 / 材质 / 字体参考
  • 示例：高饱和色块 + 3D 几何 + 毛绒 = 玩具般活力
🔑

1.2 视觉关键词定义

  • 提炼 3–4 个核心视觉形容词
  • 未来感 / 趣味性 / 高饱和 / 实验性
  • 明确「要什么」与负面清单
⭐

1.3 超级符号构思

  • 寻找承载品牌精神的图形载体
  • 如蔚来「天空与道路」、胶囊与方块

02基础系统设计

地基 · 建立品牌最核心的识别基因
🔷

2.1 标志设计

  • 图形标：极简 / 易识别 / 可缩放
  • 字标：专用字体或标准字组合
  • 组合：横竖版 / 黑白稿 / 反白稿
🌈

2.2 色彩体系

  • 主色：品牌第一识别色
  • 辅助色：区分层级 / 强调重点
  • 背景 / 中性色：阅读舒适度
🔠

2.3 字体规范

  • 标题字：有性格、吸睛
  • 正文字：易读、适合长阅读
  • 数字 / 代码：行业专用字体

03辅助图形与 IP 延展

血肉 · 丰富视觉层次，增加品牌亲和力
🧩

3.1 辅助图形开发

  • 从 Logo 提取元素裂变 / 重组
  • 设计专属纹样 / 底纹 / 边框
🧸

3.2 IP / 吉祥物（可选）

  • 具象化角色（如蓝色毛绒球）
  • 三视图 / 表情包 / 动作库
🎞️

3.3 动态视觉规范

  • Logo 演绎动画
  • 转场特效 / Loading 动画

04应用系统设计

皮肤 · 让品牌出现在所有用户接触点
💼

4.1 办公事务

  • 名片 / 信纸
  • PPT / 邮件签名
  • 工牌
🏬

4.2 环境导视

  • 门头 / 前台墙
  • 指示牌 / 贴膜
  • 沉浸式零售
📦

4.3 产品包装

  • 结构 / 材质
  • 开箱体验
  • 丝印 / 铭牌 / UI
📱

4.4 数字媒体

  • App / 网页 UI
  • 社媒头像封面
  • 公众号模板
🎁

4.5 营销物料

  • 海报 / 画册
  • 展架
  • 周边礼品

05视觉管理与培训

宪法 · 确保品牌视觉在执行中不走样
📘

5.1 编制 VI 手册

  • 规范 + 错误示范 + 应用场景图
  • 源文件交付清单
🎓

5.2 宣贯与培训

  • 向内部讲解规范背后逻辑
  • 向供应商交底工艺要求
🔍

5.3 定期审计迭代

  • 每半年 / 一年检查执行情况
  • 建立评估与迭代机制

06品牌屋 · 映射视觉系统

BRAND HOUSE × VISUAL · 结构指导视觉落地
🏠

屋顶 · 调性

  • 品牌调性与情感连接
  • 对应阶段一 视觉策略
🏛

支柱 · 核心

  • Logo / 色彩 / 字体
  • 对应阶段二 基础系统
🧱

墙壁 · 延展

  • 辅助图形与应用规范
  • 对应阶段三、四
⛰

  • 品牌识别标准
  • 对应阶段五 管理培训
UPGRADE PRINCIPLE

视觉升级 三进化

➕

增加动态层

  • 叠加「动态视觉」「数字体验」维度
🛠️

强化执行层

  • 结合 SOP，落实执行标准与工具
🖐️

五感体验扩展

  • 从视觉导向到多感官体验管理

升级 = 在原识别基因上做有约束的进化：先稳地基（Logo/色彩/字体），再演进墙壁（应用与动态），最后用 VI 宪法保证不走样。
