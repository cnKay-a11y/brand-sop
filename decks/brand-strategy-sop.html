01品牌诊断与挖掘

听诊器 · 通过系统性调研，找到市场机会点和品牌痛点
🔭

1.1 市场环境扫描

  • 宏观 PEST 分析
  • 行业规模与 3–5 年趋势
  • 3–5 个竞品 SWOT / 价格带 / 卖点
📦

1.2 内部资产盘点

  • 品牌历史与关键里程碑
  • 产品力：技术 / 供应链 / 产品线
  • 现有认知与舆情分析
🎯

1.3 用户画像描绘

  • 基础属性：年龄 / 地域 / 职业 / 收入
  • 心理图谱：价值观 / 痛点 / 爽点 / 痒点
  • 消费场景：时间 / 地点 / 情绪

02品牌顶层定位

定海神针 · 用一句话讲清核心价值，确立差异化生态位
💎

2.1 核心价值提炼

  • 功能性价值：解决具体问题
  • 情感性价值：心理满足
  • 社会性价值：群体身份
📌

2.2 品牌定位陈述

  • 目标人群 + 核心需求
  • 独特卖点 + 信任背书
  • 公式：为「人群」提供「卖点」，因为「愿景」
🧬

2.3 品牌个性设定

  • 人格化：极客 / 暖男 / 叛逆者 / 智者
  • 调性关键词：鲜明 · 年轻 · 富有节奏

03品牌话语体系构建

扩音器 · 把定位翻译成用户听得懂、记得住的语言
🔤

3.1 命名与 Slogan

  • 品牌名：易读 / 易记 / 可注册
  • 主 Slogan：传递核心价值
  • 副 Slogan：营销行动口号
📖

3.2 品牌故事

  • 起源故事：为什么创立
  • 产品故事：攻克什么难关
  • 用户故事：谁因此改变生活
🗣️

3.3 传播话术规范

  • 对外官方口径
  • 客服话术
  • 社媒文案风格指南

04品牌营销策略规划

作战图 · 制定落地打法，确保声量转化为销量
🚀

4.1 产品策略

  • 爆品打造
  • 引流 / 利润 / 形象款矩阵
💰

4.2 价格策略

  • 定价逻辑
  • 促销机制
🛒

4.3 渠道策略

  • 线上：天猫 / 抖音 / 私域
  • 线下：门店 / 快闪 / 展会
📅

4.4 推广节奏

  • 预热：悬念 + 种子用户
  • 爆发：发布会 + KOL
  • 长尾：口碑 + 会员运营
